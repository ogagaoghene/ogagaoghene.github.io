About Me

I always thought my dream was to become an industrial chemist for an environmental impact assessment laboratory analyzing samples collected from polluted environment by Oil prospecting multinational companies. It seemed like an appropriate dream given my B.Sc in Industrial Chemistry. I gravitated toward quality control positions. But, I kind of felt the need to learn something different when I was serving as a corper as a compulsory national service for young graduates in my country. I was posted to serve in an IT Firm where I took interest in the innovative world of telling computers to solve problems.

And that's when it happened; the spark I was missing ignited the instant I clicked play on my first Java Tutorial video. Since then, I can't get enough. Full-Stack web development calls to all my passions; it incorporates creativity and problem solving.

My specialties include quick learning new skills and programming languages, problem solving, domain driven design, responsive design principles, website optimization, and the Model View View Model(MVVM) and Model View Controller(MVC) methods for organizing code. So far I have Ruby, Rails, JavaScript, HTML, CSS, React, Redux, Bootstrap, Jest, RSpec, Python, Java, PostgreSQL, MySQL, ExpressJS, Cloud DevOps and Git/GitHub under my belt. I'm still enthusiastically grabbing onto any other programming languages, frameworks, or principles I can integrate into coding the web in my head.
